# ZET - Digital Solutions

A modern React + Vite website for ZET, a forward-thinking IT company.

## Setup

Install dependencies:
```bash
npm install
```

## Development

Start the dev server on http://localhost:3000:
```bash
npm run dev
```

## Build

Build for production:
```bash
npm run build
```

Preview the build:
```bash
npm run preview
```

## Structure

```
src/
  main.jsx        # React entry point
  App.jsx         # Main app component
  index.css       # Global styles
index.html        # HTML template
vite.config.js    # Vite config
package.json      # Dependencies
```

## Features

- Hero section with call-to-action buttons
- Services showcase (UI/UX, Digital Marketing, Web Dev, AI/Automation)
- About section
- Features section
- Contact form with demo alert
- Responsive design with mobile menu toggle
- CSS Grid and Flexbox layouts
- Sticky header navigation

All content is now in React components with proper structure and error fixes applied.
